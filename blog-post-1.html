Web razvoj: osnove front-enda Web Development: Front-end Basics

Kratak pregled kako nastaje moderna web stranica i šta znači “dobar” front-end. A short overview of how modern websites are built and what makes a “good” front-end.

1) HTML – struktura 1) HTML – structure

HTML je kostur stranice: naslovi, paragrafi, liste, sekcije i forme. Dobra struktura (semantički tagovi poput header, main, section, footer) pomaže i korisnicima i pretraživačima, a posebno pristupačnosti. HTML is the skeleton of the page: headings, paragraphs, lists, sections, and forms. Good structure (semantic tags like header, main, section, footer) improves usability, SEO, and especially accessibility.

2) CSS – izgled i raspored 2) CSS – style and layout

CSS daje vizuelni identitet: boje, tipografiju, spacing, kartice, animacije. Za raspored se najčešće koriste Flexbox i Grid. Responsivnost znači da stranica lijepo radi na mobitelu, tabletu i desktopu, bez “lomljenja” layouta. CSS defines visual identity: colors, typography, spacing, cards, and animations. Layout is typically built with Flexbox and Grid. Responsiveness means the page works well on phones, tablets, and desktops without breaking.

3) JavaScript – interaktivnost 3) JavaScript – interactivity

JavaScript dodaje ponašanje: validacija forme, dinamički elementi, slajderi, fetch podataka i slično. Čak i kada se (kao na ovoj stranici) dio stvari radi CSS-only, JS je često “alat za završni korak”. JavaScript adds behavior: form validation, dynamic elements, sliders, data fetching, and more. Even when some features are CSS-only, JS is often the tool for the final, polished behavior.

Zaključak Conclusion

Najbolji način učenja web razvoja je kroz male projekte: portfolio, blog, to-do, mini galerija. Svaki projekat te natjera da razmišljaš o detaljima: struktura, dizajn, pristupačnost i korisničko iskustvo. The best way to learn web development is by building small projects: a portfolio, a blog, a to-do app, a mini gallery. Each project forces you to think about details: structure, design, accessibility, and user experience.
